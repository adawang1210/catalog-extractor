## 0 病灶

M5-2 的 B 訊號(`core/m3_scan.py:231 icon_sus`):色樣面積 `a < 0.05×doc 中位` → 判疑似
版面小圖形、降級進佇列。doc 中位來自 `doc_icon_stats`(m3_scan.py:209)=**spec 頁全部
色樣面積的中位數**。

單系列型錄每張 spec 頁只有少數真色樣、卻有醒目情境照;`extract_swatches` 把情境照抽成
大面積色樣。當情境照占色樣族群比例夠高(≥半),`statistics.median` 被撐到大圖區,真小
色樣掉到 `0.05×中位` 門檻下 → **系統性誤殺真色樣**。

失敗方向錯邊:M5-2 哲學=錯殺成本僅複查、絕不毀資料;但此處錯殺的是**真色樣**(降級=該色
樣上的碼落佇列/失綁),不是 icon。此票=目前最大已標價修理票(解鎖 B 類最大檔 Uniche)。

## 2 設計案(步 2)

### 2.1 選定:med_ex —— 排除情境照後重算 doc 中位

`doc_icon_stats` 的中位改為:**排除面積 ≥ `M52_BIG×頁面積` 的大圖(情境照)後,對剩餘小簇
色樣面積取檔級中位**;小簇空(全大圖)→ 退回全體中位(保存原行為,如 41Zero42/MILANO70)。

`icon_sus` 兩臂(B `a<0.05×med`、A' `rep∧a<0.5×med`)判定式不變,僅所餵中位改為 med_ex。
per-doc、零形狀(純面積比與檔內統計,無絕對尺寸)。

**約束逐條核對:**
- **per-doc**:med_ex 為檔級統計。✓
- **零形狀**:`M52_BIG` 為頁面積比、med 為檔內統計,無任何絕對尺寸/頁數形狀條件。✓
- **失敗方向=寧可少殺 icon**:med_ex ≤ med **單調**(移除面積高端,中位只降不升)⇒ `icon_sus`
  兩臂門檻只降 ⇒ 降級只減不增 ⇒ **零新增真色樣誤殺**(硬約束),殘餘風險僅「少殺 icon」=
  icon 誤入,由 A' 臂 + M5-2 既有降級鏈 + 下游綁定兜底(§2.2)。✓
- **新常數宣告+校準+凍結**:見 §2.3。✓

### 2.2 失敗方向與 icon 洩漏帳(送審重點)

med_ex 只會「關掉」既有降級,**永不開新降級**。因此:
1. 對真色樣:只可能從「被誤殺」→「救回」,不可能新誤殺。硬約束滿足=構造性保證,非統計性。
2. 對真 icon:med_ex 在情境照撐高檔可能少殺個別 icon(洩漏)。此為**預授權的可接受方向**;
   洩漏 icon 續走 A' 臂(同位重複型 icon 仍 `a<0.5×med_ex` 被抓)與下游 x 對齊/塊綁,最終
   若誤綁到碼=落既有 needs_review,不毀資料。
3. **非全面救援之證據**:正確殺 icon 母體 145 碼(Stonetalk/Woodtouch/next/PIPO)在 med_ex
   下 dm_ex=dm_now 完全保全(這些檔無情境照撐高、med_ex≈med)。→ 修理有分離、非把 icon 一
   起放生。合成 B3/B4c 對抗案同證(極小 icon 在 med_ex 下續死)。

### 2.3 新常數 `M52_BIG` 宣告 · 校準 · 凍結

```

#   ⚠ 獨立校準:數值與 E-1 的 V9_G(0.10)同,但為本用途(中位排大圖)重新校準之判定常數,
```
